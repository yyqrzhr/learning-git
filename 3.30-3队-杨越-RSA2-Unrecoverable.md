# 1.伪随机数的确定性
## 伪随机数生成器（PRNG）不是真正的随机给定相同种子，生成相同的随机序列
Python 的 random模块使用 Mersenne Twister 算法
# 2. 种子安全性 
## 如果种子可预测，整个随机序列都可预测本题种子是 2**1337 - 1，是固定值
在实际密码学中，应使用密码学安全的随机数生成器
